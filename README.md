### how to use it
```
# git clone https://github.com/GuillaumeLeon/bot-twitch-minicli
# cd bot-twitch-minicli
# composer install
```

Make sure to change your credential in the miniclip file\
To get an Oauth token : [go here](https://twitchapps.com/tmi/)
```
# ./minicli twitch
```
